-   Descripción del proyecto:
    SportZone es un sitio web de comercio electrónico enfocado en la venta de productos deportivos como zapatillas, camisetas y accesorios. El proyecto fue desarrollado utilizando HTML5, CSS3 y JavaScript, aplicando principios de diseño responsivo y buenas prácticas de maquetación web.
